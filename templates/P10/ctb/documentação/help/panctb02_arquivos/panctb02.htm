Programa panctb02 : Definição de Regra para Exclusão de Centro de Custo por Filial

O Programa panctb02 deverá ser utilizado para a Definição da Regra para a Exclusão de Centro para uso em Determinada Filial do Sistema.

 

O seu uso é bem simples. Estando Logado na Empresa/Filial, que deseja Cadastrar regra para Exclusão de Centro de Custo, acesse o programa e defina a regra de exclusão conforme segue:

•Campo C Custo

Informe o Centro de Custo, ou tecle F3 para obter os Centros de Custo Válidos

•Campo Desc Moeda 1

É preenchido automaticamente conforme conteúdo do campo C Custo

Dica:

1. Clique na Lupa que se encontra na barra de botões para visualizar o formato de gravação da regra.

2. Clique no botão CC De/Até que se encontra na barra de botões para escolher um intervalo de Centros de Custo.

 

Procedimentos

Para incluir uma Regra de Exclusão de Centros de Custo para Determinada Filial:

1.Na janela de manutenção posicione no campo C Custo
2.Informe um Centro de Custo Válido.
3.Confira os dados e confirme a inclusão.
