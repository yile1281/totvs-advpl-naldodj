Programa panctb03 : Definição de Regra para Relacionamento dos Campos que pertencem ao Grupo 003 (Contas Contábeis) com os Campos que Pertencem ao Grupo 004 (Centros de Custo)

O Programa panctb03 deverá ser utilizado para a Definição da Regra de Relacionamento entre os Campos de Contas Contábeis e os Campos de Centros de Custo para permitir amarração e validação pelo sistema através da função u_PanVldCtb() que se encontra no programa u_PanVldCtb.prg.

 

O seu uso é bem simples. Estando Logado na Empresa, que deseja Cadastrar regra para relacionamento, acesse o programa e defina a regra de relacionamento conforme segue:

•Opção Consultar

Mostra o Conteúdo do arquivo de Relacionamento

•Opção Alterar

Disponibiliza Tela com Todas as Tabelas do Sistema que possuem campos que pertençam ao Grupo 003 e 004

Dica:

1. Clique na Lupa que se encontra na barra de botões para pesquisar uma tabela.

2. Clique no botão Filtro para filtrar as Tabelas.

3. Clique no botão Marca Todos Para Selecionar Todas as Tabelas.

4. Clique no botão Desmarca Todos Para Retitar toda Seleção.

5. Clique no botão Inverte Seleção para inverter a Seleção.

6. Após ter selecionado a(s) tabela(s), Clique no botão Executar para Definir a Regra de Relacionamento.

7. Clique no botão OK para confirmar e gravas as alterações.

8. Clique no botão CANCELAR desconsiderar as alterações feitas.

 

Procedimentos

Para incluir uma Regra de Relacionamento de Campos do Grupo 003 (Contas Contábeis) com Campos do Grupo 004 (Centros de Custo):

1.Na janela de manutenção posicione na Tabela que deseja definir a regra e "marque-a" dando dois cliques sobre o registro.
2.Clique no Botão Executar.
3.Clique no Botão Relacionar.
4.Será Disponibilizada Tela para Relacionamento dos Campos.
5.Relacione os campos da Direita (CTT) com os campos da Esquerda (CT1) informando o número correspondente.
6.Verifique se o relacionamento está correto e clique no botão OK para confirmar.
7.Após efetuar o Relacionamento da(s) Tabela(s) clique no botão OK da tela principal para gravar as alterações.
